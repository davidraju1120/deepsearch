Getting Started

The Deep Researcher Agent is an AI-powered research assistant that helps you find and analyze information from your knowledge base.

How to Use

  1. Ask Questions: Enter your research query in the main input field
  2. Enable Features: Toggle query refinement and summarization as needed
  3. Add Knowledge: Upload documents or paste text to expand your knowledge base
  4. Review Results: Analyze the research results and export them if needed

Features

  • Query Refinement: Automatically improves your questions for better results
  • Summarization: Creates concise summaries from multiple sources
  • Reasoning Explanation: Shows how the AI arrived at its conclusions
  • Export Options: Save results as Markdown or PDF files

Supported File Formats

  • PDF documents
  • Microsoft Word documents (.docx)
  • Plain text files (.txt)
  • Markdown files (.md)
